For the Phase 5 we would be looking to add a feature called start and end date to a job application. This date should be calculated based on the 'Offer Letter Received Date' and 'Employment End Date'. 

The Start date should be calculated as the number of days from the 'Offer Letter Received Date' to the current date.
The End date should be calculated as the number of days from the 'Employment End Date' to the current date.

Also, the start and end date should be displayed in the job card in the format of dd/mm/yyyy.

If the application status is not 'Interviewing' or 'Offer Letter Received' or 'Employment End Date' is not available, the start and end date should not be displayed.

Then the job cards should be sorted based on the start date and end date

Then there should be a feature that allows you to build these into a cv or something of sort, then we can also make it such that it can download a cv in pdf format.
